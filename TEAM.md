## Team Norms
- **Communication:** Discord group chat for daily updates.
- **Response time:** All members respond within 12 hours (2 hours on deadline days).
- **Meetings:** Meet twice per week for 30–45 minutes.
- **Decision making:** Majority vote; Repo Lead decides if tied.
- **Task ownership:** Every GitHub issue must have one assigned owner.
- **Definition of done:** Feature runs locally, meets acceptance criteria, and README updated if needed.
- **Conflict/blockers:** If blocked for more than 24 hours, notify the Backlog Lead.
